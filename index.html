DATA EXPLORATION

Exploring Sales Data from a Coffee business and creating Visualizations using tableau

Summary

Explored sales data from a coffee vending machine to gain insights into product performance, revenue trends, pricing strategies, and customer payment behaviors. The data set includes individual coffee sales with timestamps, prices, and payment methods.

Key Insights:

  • Identified Top-selling beverages based on total units sold.
  • Revealed Highest revenue-generating beverages through total sales revenue.
  • Analyzed Monthly sales and revenue trends to understand seasonality.
  • Calculated Average price per beverage.
  • Identified Peak sales hours.
  • Sales breakdown by payment method (cash vs card) analyzed for customer preferences.

Data cleaning

Cleaning data using SQL

Summary

This project involved cleaning and standardizing a raw dataset (animal_data_dirty) containing wildlife observations. The dataset included information such as animal type, weight, body length, location coordinates, observation date, and other metadata. Data quality issues included inconsistent naming, incorrect data types, unrealistic values, and missing entries.

Key Steps and Fixes:

  • Schema Fixes: Renamed columns to standardized snake_case for readability and consistency.
  • Duplicate Removal: Identified and removed duplicates using ROW_NUMBER() partitioning.
  • Standardizing Categorical Values: Corrected typos and normalized values in animal_type and Country.
  • Handling Missing & Invalid Values: Converted blanks to NULLs, removed unusable rows, and cleaned coordinates.
  • Data Type Conversions: Changed key columns from text to numeric/date types.
  • Outlier Correction: Added nulls to unrealistic values (for instance, Lynx weighing 171 kg).
  • Imputation: Inferred animal_type using known weight and length ranges.
  • Dropped Columns: Removed empty columns like animal_code.
  • Date Formatting: Standardized and converted Observation date into DATE type.
